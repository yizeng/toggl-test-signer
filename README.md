# toggl-test-signer

## How to Run?

### Native (go 1.21.4)

Run `make run` and then navigate to <http://localhost:8080>

### Docker

Run `make docker` and then navigate to <http://localhost:8080>

## API Endpoints

### `POST    /api/v1/admin/verify-signature`

### `POST    /api/v1/users/sign-answers`
